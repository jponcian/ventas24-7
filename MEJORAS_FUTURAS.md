# Plan de Mejoras Propuestas - Ventas 24/7

Este documento contiene las recomendaciones para elevar la calidad, funcionalidad y estética de la aplicación.

## 1. Experiencia Visual y Marca (Aesthetics)
- [ ] **Rebranding "Ventas 24/7":** Implementar una identidad visual moderna con una paleta de colores premium (azul oscuro y acentos dorados/vibrantes) y tipografía *Outfit*.
- [ ] **Modo Oscuro:** Soporte nativo para tema oscuro.
- [ ] **Micro-interacciones:** Animaciones con Lottie para feedback visual (ventas exitosas, errores, carga de datos).

## 2. Funcionalidades de Venta y Cliente
- [ ] **Recibos por WhatsApp:** Envío de comprobantes de venta formateados por mensajería.
- [ ] **Impresión Térmica:** Soporte para impresoras Bluetooth (58mm/80mm).
- [ ] **Gestión de Cuentas (Fiados):** Notificaciones de recordatorio de pago y estado de cuenta detallado para clientes.

## 3. Gestión Inteligente e Inventario
- [ ] **Alertas de Vencimiento:** Panel de productos próximos a vencer (7/15/30 días).
- [ ] **Predicción de Reabastecimiento:** Estimación de agotamiento de stock basada en ritmo de ventas.
- [ ] **Optimización de Escáner:** Mejora del rendimiento del escáner en entornos de baja luz.

## 4. Seguridad y Rendimiento
- [ ] **Acceso Biométrico:** Login con Huella/FaceID.
- [ ] **Sincronización Automática:** Background sync para ventas realizadas en modo offline.
- [ ] **Backups en la Nube:** Respaldo automático de la base de datos para prevenir pérdida de datos.
